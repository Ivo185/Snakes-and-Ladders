# 🐍 Змии и Стълби — Образователна настолна игра

Дигитална версия на класическата игра "Змии и Стълби", комбинирана с викторина по история на България. Играта е разработена на Python с графичен интерфейс чрез `tkinter` и поддържа до 6 играча, звукови ефекти, анимации и интеграция с micro:bit за хвърляне на зара.

---

## 🎮 Функционалности

- **2 до 6 играча** на един компютър
- **Въпроси по история** на България, организирани по трудност (Лесно / Трудно / Всички)
- **Таймер** за всеки въпрос (конфигурируем)
- **Змии и стълби** на класическите позиции в полето (100 клетки)
- **Звукови ефекти** — зар, фойерверки, победа и поражение
- **Анимация на фойерверки** при победа
- **Micro:bit интеграция** — разклати micro:bit за хвърляне на зара (опционално)
- **Добавяне на въпроси** чрез отделно приложение `AddQuestions`
- **Пълноекранен режим** с адаптивно мащабиране

## 🚀 Стартиране

### Варант 1: Стартиране от `.exe` (препоръчително)

1. Свалете последния release (`Змии_и_стълби_v2.zip`)
2. Разархивирайте
3. Стартирайте `Game.exe`
4. *(По желание)* Стартирайте `AddQuestions.exe` за добавяне на въпроси

### Вариант 2: Стартиране от код

**Изисквания:**
```
Python 3.10+
pip install pillow pygame pyserial
```

**Стартиране:**
```bash
cd "V2 stable"
python Game.py
```

## 🎲 Как се играе

1. Изберете **брой играчи** (2–6)
2. Изберете **трудност на въпросите** (Лесно / Трудно / Всички)
3. При кацане на специално поле — отговорете на въпрос:
   - ✅ Верен отговор → качвате се по стълба / не слизате по змия
   - ❌ Грешен → слизате по змия / не се качвате по стълба
4. Първият играч, достигнал поле 100, печели!

## 🐛 Micro:bit интеграция (само V2)

1. Качете `Micro-bit code.py` на вашия micro:bit чрез [MakeCode](https://makecode.microbit.org/) или mu-editor
2. Свържете micro:bit към компютъра чрез USB
3. Стартирайте `microbit_test.py` за да проверите връзката и порта
4. В играта — разклатете micro:bit вместо да кликате за хвърляне на зара

## ❓ Формат на въпросите (`Въпроси.md`)

Всеки въпрос е на отделен ред в следния формат:

```
Трудност | Въпрос | Отговор | Секунди
```

**Пример:**
```
Лесно | Кога е Покръстването на българите? | 864 г. | 60
Трудно | През коя година византийците превземат Преслав? | 971 г. | 45
```

За добавяне на въпроси използвайте `AddQuestions.exe` / `AddQuestions.py`.
